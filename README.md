# CEH Study Resources

This repository contains study materials for the **Certified Ethical Hacker (CEH)** exam.
It includes practice dumps, Anki flashcards, and structured notes covering **all 20 CEH modules**.

The purpose of this repository is to help with **exam preparation, revision, and practice questions**.

# Study Materials

## Flashcard.colpkg

`Flashcard.colpkg` is an **Anki flashcard deck** containing practice questions and answers for CEH topics.

The flashcards are designed to help reinforce knowledge through **spaced repetition learning**.

### How to Import into Anki

1. Install **Anki**

   [https://apps.ankiweb.net/](https://apps.ankiweb.net/)

2. Open Anki.

3. Click:

```
File → Import
```

4. Select the file:

```
Flashcard.colpkg
```

5. The flashcards will automatically appear in your deck list.

You can now start practicing questions and reviewing them regularly.

## Dumps Folder

The `Dumps/` folder contains **practice exam questions and study dumps** collected from various sources and versions of the CEH exam.

These files include question banks for:

* CEH v11
* CEH v12
* CEH v13

They can be used for:

* Practice exams
* Understanding common question formats
* Identifying frequently tested topics
